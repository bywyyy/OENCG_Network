该文件夹用于存放游戏定义文件。

form.xml文件定义游戏基本信息，游戏玩家个数，各玩家资源数量，以及联盟形成的最少资源数量(majority);

coalitions.xml文件中枚举了联盟的所有可能形式，以及每个联盟可获得的收益；

talents.xml文件设置了每个玩家隐藏信息值，即游戏中的Cost

knowlege.xml定义了玩家隐藏信息对其他玩家的可见性。

自定义自己的游戏需要在以上四个文件中分别进行修改，强烈建议进行自定义后将修改后的文件保存在一个单独的文件夹下面，以便以后使用。

3players文件夹5players文件夹下便定义了两个示例游戏。如需要进行五个玩家之间的游戏，只需要用5players文件夹下的所有文件覆盖掉domain-repository根目录下对应的四个文件即可。

自定义游戏之前请务必了解根目录下的form.xml, coalitions.xml, talents.xml, knowledge.xml文件的作用以及每个文件描述的信息(每个文件都有注释)，再进行更改，以免出现新建房间无反应、失败的情况。
